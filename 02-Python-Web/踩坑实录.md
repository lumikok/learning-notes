# 踩坑实录

## 学习阶段的疑惑点

### 文件启动方式

```bash
fastapi dev
```

默认执行逻辑：

1. 先找 `main.py`
2. 找不到 → 找 `api.py`
3. 找不到 → 找 `app.py`
4. 找不到 → 找 `index.py`
5. 都找不到 → 报错

想运行别的，在后面加上文件相对路径（此处，发挥 `Tab` 键神力）

**常见状态码要记一下的哈**：

- `200`：正常
- `422`：参数格式 / 类型校验失败，接口拒绝接收数据


### 严格模式

Pydantic 自动类型转换有好有坏，可以开启严格模式，让类型提示升级为类型限定。
