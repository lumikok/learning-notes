## 🧭从迷宫说起（再次）

还是那个迷宫，但这次你想找的**不是随便一条跑，而是最短的出口路径**。

DFS 会一条路走到黑，可能绕很远。现在，我们换一种思路：**你从入口开始，先走一步能到的所有位置，再走两步能到的所有位置……** 这样第一次遇到出口时，走的步数显然是最少的。

这就是广度优先搜索（BFS）的核心思想：**像水波一样，一层一层往外扩散**。

总结：BFS 会先访问离起点最近的所有节点，然后再访问稍远一点的节点，依次推进。常用于解决**边权相同的最短路径问题**。

**何为边权？**
就是路径代价，形象来说类似于边长、距离。

## 🧱BFS 的数据结构：队列

从上面的例子中，我们不难看出，BFS 的特征就是**逐层搜索**。

很明显，这意味着需要先处理较近的节点，这与**队列（先进先出）** 的特征相符。因此，BFS 就是依据队列实现的。

## 🌲树上的 BFS（层序遍历）

在树上执行 BFS，就是**层序遍历**：从上到下，从左到右逐层访问。

### 二叉树层序遍历

```cpp
struct TreeNode {
    int val;
    TreeNode *left;
    TreeNode *right;
    TreeNode(int x): val(x),left(nullptr),right(nullptr) {}
};

void bfs(TreeNode *root) {
    if (root == nullptr) return;
    queue<TreeNode*> q;   // 定义节点指针类型的队列
    q.push(root);
    while(!q.empty()) {
        TreeNode* node = q.front();
        q.pop();
        cout << node->val << " ";  //访问当前节点
        // 在处理当前节点的同时，将下一层的节点按顺序入队
        if(node->left) q.push(node->left);
        if(node->right) q.push(node->right);
    }
}
```

**输出顺序**：这里注意，BFS 在树上的输出与 DFS 不同：BFS 是“按层”，DFS 是“按深度优先”。
