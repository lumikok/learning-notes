# P 1443 马的遍历

## 原题链接

https://www.luogu.com.cn/problem/P1443

## 题目简述

n×m 棋盘，马从 (x,y) 出发，求到每个点的最少步数，到不了输出 -1。
以象棋为背景。
## 核心思路

- **标签**：BFS，队列；
- **思路**：马的移动方向有八个，距离数组初始化 -1（未到达），初始点设为 0。

**经典 BFS 目标点最短路径**

## 关键代码

**程序链接**。

```cpp
int dx[] = {1,1,-1,-1,2,2,-2,-2};
int dy[] = {2,-2,2,-2,1,-1,1,-1};

// 二维坐标处理技巧
queue<pair<int,int>> q;
board[x][y] = 0;
q.push({x,y});
while(!q.empty()) {
    auto cur = q.front();
    q.pop();
    int nx = cur.first;
    int ny = cur.second;
    for(八个方向) {
        int ax = nx + dx[i];
        int ay = ny + dy[i];
        if(边界内&& board[ax][ay] == -1) {
            board[ax][ay] = board[nx][ny] + 1;
            q.push({ax,ay});
        }
    }
}
// 初始化数组的技巧
memset(board,-1,sizeof(board)) 
```

## 注意点

- 方向数组不要写错；
- 处理二维坐标类的技巧（pair）；
- 条件判断写全，别出错；
