# B 3862 图的遍历

## 原题链接

https://www.luogu.com.cn/problem/B3862#ide

## 题目简述

N 个点，M 条边的有向图，求从 i 出发能到达的最大点。

## 核心思路

- **标签**：DFS，图
- **思路**：邻接表存图，标记是否到过，保存每次的最大值。

## 关键代码

**程序链接**。

```cpp
vector<int> graph[maxn];
bool vis[maxn];
int max_num = 0; 

// DFS函数核心
vis[u] = true;

if(u > max_num) max_num = u;

for(int v : graph[u]) {
    if(!vis[v]) dfs(v);
}

遍历每个点dfs即可
```

## 注意点

- 判断大小是关键；
- 遍历每个点结束后，一定要在这里重置 `vis` 数组；
- `max_num` 初始化为当前节点；
